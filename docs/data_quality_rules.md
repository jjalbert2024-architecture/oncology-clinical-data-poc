## Rule: Valid ICD-10 Code
Severity: Critical  
Description: All oncology diagnoses must map to valid ICD-10 codes to support clinical, registry, and claims reporting.

## Rule: Cancer Stage Present
Severity: Critical  
Description: Cancer stage is mandatory for oncology treatment planning and outcome analysis.

## Rule: Diagnosis Date Not in Future
Severity: Warning  
Description: Diagnosis dates in the future indicate potential data entry or integration issues.

## Rule: Valid Patient Reference
Severity: Critical  
Description: All diagnosis records must map to an existing patient.
